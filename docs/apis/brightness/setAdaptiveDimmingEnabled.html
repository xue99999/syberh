setAdaptiveDimmingEnabled

syberh.brightness.setAdaptiveDimmingEnabled(Object object)

设置自适应调光开关。

权限

参数

Object object

属性类型必填描述
statestring是0表示关闭，1表示开启，默认为0
successfunction否成功回调
failfunction否失败回调

object.success 回调函数参数

参数

object.fail回调函数参数

参数

属性类型描述
codeString错误码
msgString错误信息

代码示例

syberh.brightness.setAdaptiveDimmingEnabled({
    state="0",
    success:function(result){

    },
    fail:function(error){
        console.log('fail: ', error.code, error.msg);
    }
});
