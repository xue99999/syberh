startPlay

播放录音：可以对录音记录进行播放。

syberh.audio.startPlay(Object object)

参数

Object object

属性类型默认值必填描述
pathstring是录音路径
successfunction否回调成功
failfunction否回调失败

object.fail回调函数

参数

属性类型描述
codeString错误码
msgString错误消息

代码示例

syberh.audio.startPlay({
  path: "/home/user/audio/20190905_1567662466.aac",
    success: function(result){
    console.log('success');    
    },
    fail: function(error){
        console.log('fail: ', error.code, error.msg);
    }
});
